Visual Studio

由微软公司出品，人称“宇宙第一IDE”，功能极强、体积极大、价格很贵。社区版免费，但需要登录微软账号方能使用。

JetBrains 系列

JetBrains成立于2000年，总部位于捷克，致力于打造世界上最强大、最高效的开发者工具，几乎为每一种流行的语言都做了对应的IDE， 因此在这里归为一个系列。
不过这家公司的IDE比 Visual Studio 还慢……

Eclipse

曾经最好用的免费Java语言IDE，不过近来有被 IntelliJ IDEA 取代的趋势。

MyEclipse

建立在Eclipse基础之上的企业级开发IDE，集成了更多的企业级开发功能。

NetBeans

一套跨平台的开发环境、工具平台、程序框架。主要支持Java和PHP、Web等技术栈。

Code::Blocks

免费的C/C++和Fortran语言的IDE。

CodeLite

CodeLite是一个开源、免费、跨平台的IDE，专门用于C、C++、Rust、Python、PHP和JavaScript （主要针对使用Node.js的后端开发人员）。

KDevelop

KDevelop是一个功能齐全、可扩展的IDE，适用于C/C++和许多其他编程语言。 支持的平台包括Linux、Solaris、FreeBSD、macOS和其他Unix版本以及Microsoft Windows。 它基于KDevelop平台（kdevplatform）、KDE框架和Qt库。它自1998年以来一直在开发中。
