SQL Server

由微软公司出品的强大数据库，分为“快速版”、“开发人员版”、“标准版”、“企业版”，其中“快速版”是免费的。

MySQL

由芬兰程序员米卡埃尔·维德纽斯于1995年开发完成。2008年，其创立的公司“MySQL AB”被“Sun”公司收购；2009年，“Sun”公司又被“Oracle”公司收购。由于Oracle收购的产品通常呈现闭源趋势， 因此米卡埃尔·维德纽斯根据MySQL开发了新的数据库MariaDB。
MySQL分为多个授权版本，其中社区版是免费的。

SQLite

用C语言打造的纯文件、无服务型的开源免费数据库，短小精悍，特别适合嵌入式和小型项目。

PostgreSQL

有着30多年历史的开源数据库。PostgreSQL的起源可以追溯到1986年，当时是加州大学伯克利分校POSTGRES项目的一部分。

Oracle

最大牌、最可靠的关系型数据库，当然体积也最大、价格也最贵。一般只有银行等对数据安全要求特别高的传统行业才会使用。

MariaDB

MySQL被Oracle收购后，其创始人米卡埃尔·维德纽斯自创了开源新分支：MariaDB，整体架构兼容MySQL。
