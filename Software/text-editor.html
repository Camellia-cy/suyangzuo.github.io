Visual Studio Code

微软公司于2015年推出的开源免费文本编辑器，使用TypeScript语言编写。经过多年的发展，已经成为世界上最为强大的文本编辑器，是许多程序员的最爱。

Sublime Text

2008年推出的跨平台文本编辑器，很多程序员都非常青睐它。

HBuilderX

专为Vue打造的文本编辑器，由 DCloud，即数字天堂（北京）网络技术有限公司（是W3C成员及HTML5中国产业联盟发起单位）打造。

记事本

你可能觉得奇怪，记事本也配放上来？主要是因为多年不变的记事本更新了，虽然功能还是一样烂，但为了提醒一下不知道的人，就这样吧……

MarkText

代码与预览一体化的Markdown编辑器，完全免费、完全本地。

UltraEdit

已诞生近30年的老字号文本编辑器，曾经的首选，但自从VSCode出现后就逐渐跟不上了，毕竟VSCode免费又好用，而这玩意儿还要交年费。

Vim

非常有名的文本编辑器，传说很多人连怎么退出它都搞不清楚。反正我没用过，自己看着办吧。
