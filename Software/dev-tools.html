MSYS2

MSYS2是一组工具和库，为开发者提供了一个易于使用的环境，用于构建、安装和运行原生Windows软件。
它包括名为mintty的命令行终端、bash、git和subversion等版本控制系统、tar和awk等工具， 甚至包括autotools等构建系统，所有这些都基于Cygwin的修改版本。

.NET

.NET平台开发必备。注意：“运行时”只能“运行”.NET软件，是给终端用户使用的；如要开发.NET软件，必须下载SDK。

Java SE Development Kit

开发Java程序必备。这里提供的是Oracle官方的 Java SDK。

Python

官方的Python开发环境。

Anaconda

Anaconda公司提供的Python环境，包含了大量的第三方库，在数据分析领域很有名。 该环境分为多个等级，其中 Anaconda Distribution 是开源免费的。
注意：Anaconda总是比官方低一个版本。
