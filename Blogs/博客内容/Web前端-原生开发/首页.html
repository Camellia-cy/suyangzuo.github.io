什么是Web前端❓

  • Web开发的主要目标是开发符合客户需求的网站、应用程序。在整个开发过程中，职责通常分为前端和后端（台前表演和幕后工作者）。

    • 前端职责  用户界面（User Interface，简称 UI），简单地说，就是用户能看到的部分，因此前端也可称为客户端。
    • 后端职责  数据的后台处理，其详情不对用户展示涉及用户隐私或知识产权，因此后端也可称为服务器端。

  • 如何成为前端开发者？

    • 前端是一个富有创造力和表现力的工作，现代前端的框架非常多，但其核心支柱就只有3个。
    • HTML 用于构筑页面内容。
    • CSS 对内容进行布局和样式化，让页面看起来更加美观、舒适。
    • JavaScript 为页面带来互动性，用户能获得更强的交互感。
