属性😮 属性😨 属性🤬

  • Web前端的属性确实有点烦人，但你可以松开拳头吗？😇

  • 属性分为3种。

    • Attribute 中文翻译就叫属性，指的是HTML标签属性。比如有如下元素： 

      <img src="./rabbit.png" alt="兔子照片" title="兔子" />

      这是一个img标签，包含3个属性HTML标签属性，即3个attribute：

      • src  图片来源。
      • alt  图片加载失败时显示的替换文本。
      • title  鼠标指向图片时弹出的信息。

      因此，在JavaScript中，要修改图片的来源，可以用 setAttribute() 方法。

    • Property 中文翻译也叫属性，但指的是样式的属性style的属性。比如有如下元素： 

      <div style="color: red; font-size: 1rem; margin: 10px"></div>

      这是一个div标签，包含1个属性HTML标签属性，即1个attribute：

      • style  div元素的样式。

      而style中包含3个属性样式属性，即3个property：

      • color  文本颜色。
      • font-size  字体尺寸。
      • margin  外边距。

      因此，在JavaScript中，要修改样式中的文本颜色，可以用 setProperty() 方法。

    • JavaScript 对象属性，是 JavaScript 中对象object的属性，类似于其它面向对象语言中的字段，在这里就不展开阐述了。

  • 来总结一下吧！🍭

    • HTML标签属性 Attribute
    • style样式属性 Property
    • 只是因为中文翻译的问题，都叫属性，其实英文不一样。
