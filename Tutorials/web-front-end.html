伯纳斯·李
Timothy John Berners-Lee

详细生平

蒂莫西·约翰·伯纳斯-李，英国计算机科学家，万维网、HTML的发明者。
1989年夏天，蒂姆成功开发出世界上第一个Web服务器和第一个Web客户端。
1989年12月，蒂姆为他的发明正式定名为 World Wide Web。
1991年5月，万维网在Internet上首次露面，立即引起轰动，获得了极大的成功，并被广泛推广应用。
布兰登·艾克
Brendan Eich

详细生平

布兰登·艾克，1961年7月4日出生于美国加州，美国程序技术专家与企业家，JavaScript主要创造者与架构师，曾任Mozilla公司的首席技术官，并曾短暂担任首席执行官。
1995年4月，任职于网景期间，为网景浏览器开发出JavaScript，之后成为网页浏览器领域应用最广泛的脚本语言之一。
1998年，艾克协助成立Mozilla.org，2003年在美国在线决定结束网景部门营运后，艾克协助成立了Mozilla基金会。
W3 Schools
最知名的在线教程网站，每个月约6000万访问量。拥有多种语言的教程，其中Web开发最为著名。
MDN Web Docs
由现代Web技术标准的创建者：Mozilla公司（也是火狐浏览器的拥有者）于2005年创建，专注于Web开发相关的资源和教程。
Learn CSS Grid
专门讲解CSS网格布局的网站，对网格布局做了全面的概述，图文并茂，直观易懂。
CSS Reference
罗列了所有的CSS属性，以及每个属性在不同值下的效果图，非常直观。
MarkSheet
一个学习Web前端的综合网站，章节清晰明了。
JavaScript.Info
从俄罗斯最大的 JavaScript 教程和学习平台 learn.javascript.ru 演化而来，有简体中文版。教程覆盖范围广，每个条目细则都有案例代码可供参考。
CSS-Tricks
专门讲解CSS高端技巧的网站，拥有各路高手炫酷的CSS案例。建议拥有扎实的Web开发基本功后，学有余力的话再去深造。
