丹尼斯·里奇
Dennis Ritchie
丹尼斯·里奇（1941年9月9日-2011年10月12日），C语言之父，UNIX之父。曾担任朗讯科技公司贝尔实验室下属的计算机科学研究中心系统软件研究部的主任一职。1978年与布莱恩·科尔尼干（Brian W. Kernighan）一起出版了名著《C程序设计语言（The C Programming Language）》。此书已翻译成多种语言，被誉为C语言的圣经。2011年10月12日（北京时间10月13日），丹尼斯·里奇去世，享年70岁。
详细生平
微软官方文档
由微软公司发布、维护，面向C++、C语言、汇编程序的在线文档。
微软公司的文档以信息详实著称。但该文档有一定的阅读门槛，初学者看起来会比较晦涩。
Learn-C.org
一个免费非盈利的教程网站，仍在不断建设完善中；教程分为基础部分和高级部分。
Tutorials Point
创建于2006年的在线教程网站，条目清晰，案例详实，对各种概念解释得也很到位。
Programiz
一个充满激情的小团队创建的学习资源网站。包含了传统的C语言学习章节。每个章节都有 C++/C 语言的代码范例。
Javapoint
由印度人 Sonoo Jaiswal 创办。别被网站名称骗了，除了Java，网站中还有大量其它技术栈的教程。
C语言页面中，除了基础教程，还介绍了很多编写C程序的高级技巧。
GeeksforGeeks
由印度人 Sandeep Jain 创办。
教程采用了模块化设计，除了传统章节外，还有小技巧、进阶教程等。每个章节都配有 C/C++ 的代码演示。
C语言中文网
C语言中文网成立于2012年初，由站长 “严长生” 在大三寒假期间创办，致力于分享优质编程教程，帮助对编程感兴趣的读者。
