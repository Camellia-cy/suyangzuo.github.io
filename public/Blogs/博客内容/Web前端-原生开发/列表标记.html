列表标记类型

自定义预设标记类型

每个列表项前面都有个标记，CSS提供了很多预设标记类型。如果我们想用自定义的标记类型，CSS属性list-style-type的值要用字符串类型： 

      
        <ul class="列表">
          <li class="列表项">谷歌</li>
          <li class="列表项">必应</li>
          <li class="列表项">百度<</li>
          <li class="列表项">鸭鸭跑</li>
        </ul>
      
    
      
        .列表项 {
          list-style-type: "😄";
        }

HTML 字符集

HTML 提供了非常多的字符集供使用，类似😂等表情符号，或者 ☂ 等符号，都属于HTML 字符集，其中每个字符都对应一个编码。编码一般有两种形式：

  • 10进制，用于HTML
  • 16进制，用于CSS

很多网站都可以查到这些编码，比如在W3 Schools上：

假设我们需要使用🥶表情，它的用法是：

  • HTML：&#129398;
  • CSS：\1f976

当然，从可读性的角度来说，直接复制符号也可以。因此，以下两种写法效果是相同的： 

        
          .列表项 {
            list-style-type: "😄";
          }
  
          /* 或者 */
  
          .列表项 {
            list-style-type: "\1f604";
          }

注意：HTML 字符集包含了好几个种类，其中Emoji 表情是无法用color来修改颜色的，但是其它字符可以。

自定义标记颜色

假设我们需要将标记改成浅蓝色，可以用color属性对Emoji 表情无效： 

      
        .列表项 {
          list-style-type: "✍";
          color: lightblue;
        }

问题来了，标记的颜色是变了，但是列表项文本的颜色也变了，这往往不是我们想要的结果。如果只想修改标记的颜色该怎么办？这里我们就要用到::marker伪元素： 

      
        .列表项 {
          list-style-type: "✍";
        }
        
        .列表项::marker {
          color: lightblue;
        }

通过::marker伪元素可以单独定制标记样式。
