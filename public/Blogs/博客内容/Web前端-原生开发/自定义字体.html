字体版权

很多人不重视字体的版权，甚至不知道字体是有版权的，这可能会导致版权方发来律师函甚至是直接索赔。在中国，最著名的案例就是 2007 年北大方正起诉暴雪在其由中国大陆代理运营的网络游戏《魔兽世界》中，未经授权使用了 5 款方正字体，并索赔的案件。相关信息可在国家版权局官网查到。因此，建立字体的版权意识是非常重要的。

字体类型

衬线字体Serif、无衬线字体Sans Serif

下图是衬线字体和无衬线字体的样本：

两种字体的区别很明显。衬线字体具有笔划末端的特征，无衬线字体没有这种特征。

  • 衬线字体：适用于古典、传统、有历史气息的场景。
  • 无衬线字体：适用于于现代、科技、创造力的场景。永恒是一个专注于解决现代编程开发、设计的网站，因此基本上都是用无衬线字体

等宽字体Monospace、变宽字体

下面是用两种字体编写相同的代码，区别一目了然：

等宽字体：const element = document.querySelector("#username")
变宽字体：const element = document.querySelector("#username")
等宽字体：llllllllllWWWWWWWWWW
变宽字体：llllllllllWWWWWWWWWW

  • 等宽字体：所有字符宽度相同，适合编写代码。
  • 变宽字体：字符宽度有宽有窄，适合文章、书信。
