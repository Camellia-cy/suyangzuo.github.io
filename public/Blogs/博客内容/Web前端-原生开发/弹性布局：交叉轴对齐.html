交叉轴简介

什么是交叉轴？

主轴与弹性方向平行；交叉轴与弹性方向垂直。

弹性方向决定了主轴与交叉轴的方向。

提醒

本专题将以默认的弹性方向为基础，即主轴为行，交叉轴为列。

为什么要单独讨论交叉轴？

在交叉轴方向，有时会出现多个行；行本身、行内部的元素如何布局，分别对应align-content和align-items，这是很多人初学弹性布局最困惑的地方。因此，本专题将专门讨论交叉轴的对齐方式。

弹性如何体现？

一个元素如果使用了弹性布局，其内部的直接子元素会成为一个弹簧，意味着可以收缩和膨胀；区别在于：默认状态下会收缩而不会膨胀。

不管增加多少元素，它们都会尽可能地收缩在弹性容器内部，就像很多弹簧被压缩着挤在一个盒子里，这就是默认状态下会收缩的特性。

维持弹簧原本的尺寸

弹性布局内部的直接子元素默认会收缩。如果希望维持它们原本的尺寸，该怎么做呢？方法有两种：

  • 设置弹性收缩为0 
              
            弹性容器的直接子元素 {
              /* flex-shrink 默认为 1 */
              flex-shrink: 0;
            }

    设置了flex-shrink: 0后，弹性容器的直接子元素不会再收缩，而是维持原本的大小。不过，这种方式有时会导致子元素溢出容器。

  • 允许换行，一行放不下就放到下一行 
              
            弹性容器 {
              /* flex-wrap 默认为 nowrap */
              flex-wrap: wrap;
            }

    设置了flex-wrap: wrap后，一行放不下的元素，会自动放到下一行，这就是多行的由来。可以看出，这几行是从上往下生成的，意味着多行生成于交叉轴。

从以上方法可以看出，只有设置flex-wrap: wrap，交叉轴才会出现多行的情况。省略flex-wrap或者设置为flex-wrap: nowrap，交叉轴就只有一行。
