浏览器引擎

浏览器引擎概述

  • Webkit：苹果公司开发，用于Safari以及以及其它苹果系统上的浏览器。
  • Blink：谷歌公司开发，从Webkit分支而来，用于Chrome、Edge等浏览器。

    小知识

    苹果公司开发了Webkit引擎，谷歌最早也是使用这个引擎开发浏览器的。后来，谷歌根据自身需求，从Webkit分支出来，开发出了Blink引擎，加上V8 JavaScript 引擎，开发了Chromium项目。Chromium是一个免费开源的浏览器项目但不是浏览器，任何人都可以用这个项目来开发浏览器。

    基于Chromium项目的浏览器，现在已经占据了绝大部分市场；最有名的就是Chrome和Edge。由于其核心的复杂历史，这类浏览器有很多称呼：

    • Webkit 核心其实它现在应该特指苹果系统的浏览器
    • Blink 核心
    • Chromium 核心

    以上说法，其实都是同一个意思，只不过不同的人从不同的角度称呼它们而已。

  • Gecko：Mozilla公司开发，用于Firefox浏览器。

    提醒

    Firefox是这几个浏览器中唯一一个在中国大陆有代理的，因此也分成了国际版和国内版很多人都不知道有这回事。但凡电脑软件用的多、用的久，有经验的都知道，只要分了国际版和国内版，那国内版定然没有好事，可能出现的情况有：

    • 国际账号和国内账号不互通
    • 国内无法使用部分软件功能或服务如Adobe系列
    • 国内版无法获取特定特性，比如防广告扩展如Firefox

    Firefox在国内由北京谋智公司代理，可以打开火狐浏览器的帮助菜单，如果有北京谋智字样，那就是国内版。

    关于信息中包含了北京谋智，这就是国内版。

    关于信息中只有Mozilla，这就是国际版。

    我们肯定要使用国际版，但很多人往往在不知情的情况下下载了国内版，如何区分它们？

    • 国际版：https://www.mozilla.org/zh-CN/firefox/，域名是mozilla.org
    • 国内版：https://www.firefox.com.cn/，域名是firefox.com.cn

    特别提醒：如果直接访问国际版域名，会被重定向到国内版域名！

    直接访问mozilla.org，结果被自动重定向到firefox.com.cn，很多人都被这瞒天过海的招数骗了。因此，要下载国际版，需要直接访问国际版的Firefox 页面，而不是主页。
