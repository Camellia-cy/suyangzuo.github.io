属性原创

作者： 苏扬

日期： 2023年6月19日

属性😮 属性😨 属性🤬

  • Web前端的属性确实有点烦人，但你可以松开拳头吗？😇

  • 属性分为3种。

    • AttributeHTML 中文翻译就叫属性，指的是HTML元素属性。比如有如下元素： 

      <img src="./rabbit.png" alt="兔子照片" title="兔子" />

      这是一个img元素，包含3个属性HTML元素属性，即3个attribute：

      • src  图片来源。
      • alt  图片加载失败时显示的替换文本。
      • title  鼠标指向图片时弹出的信息。

      因此，在JavaScript中，要修改图片的来源，可以用 setAttribute() 方法。

    • PropertyCSS 中文翻译也叫属性，但指的是CSS样式属性一种特征。比如有如下元素： 

      <div style="color: red; font-size: 1rem; margin: 10px"></div>

      这是一个div标签，包含1个属性HTML元素属性，即1个attribute：

      • style  div元素的样式。

      而style中包含3个属性样式属性，即3个property：

      • color  文本颜色。
      • font-size  字体尺寸。
      • margin  外边距。

      因此，在JavaScript中，要修改样式中的文本颜色，可以用 setProperty() 方法。

    • PropertyJavaScript  中文翻译同样是属性，是 JavaScript 中对象object的成员，在这里就不展开阐述了。

  • 来总结一下吧！🍭

    • HTML元素属性 Attribute
    • CSS属性 Property
    • JavaScript对象的成员 Property
    • 只是因为中文翻译的问题，都叫属性，其实英文不一样。
