Windows官方终端简介

Windows官方终端有哪些？

如 ·图1· 所示，Windows的官方终端目前有2个：

  • ①Windows 控制台主机传统，也就是我们常说的 命令提示符cmd。
  • ②Windows 终端现代

什么是命令提示符？

如 ·图2· 所示，命令提示符是大多数Windows操作系统中可用的命令行解释器应用程序，用于执行输入的命令。

提醒

命令提示符有时被错误地称为“DOS提示符” 或 MS-DOS。命令提示符是一个Windows程序，它模拟MS-DOS中的许多命令行功能，但它不是MS-DOS。

命令提示符有什么缺点？

  • 界面丑陋
  • 缺乏自定义选项
  • 太过古老
  • 功能不够强大

什么是Windows终端？

如 ·图3· 所示， Windows 终端 首发于2019年2月，由微软公司研发，是一款面向命令行用户的新型、现代化、功能丰富、高效的终端应用程序。它包括Windows命令行社区最常要求的许多功能，包括对选项卡、富文本、全球化、可配置性、主题化和样式化等的支持。

Windows终端有什么优点？

  • 免费开源
  • 微软公司背书，GitHub社区高度参与，性能、支持有保障
  • 标签化，可拆分
  • 高度自定义
  • 可聚合各种命令行工具
  • 支持富文本

提醒

即使是现代化的 Windows 终端，调用的也是其它的命令行工具，而这些命令行工具默认都只能显示枯燥的文本。但是，有的命令行工具可以高度自定义，既可以产生丰富的视觉效果，又能提供丰富的信息，比如PowerShell，网络上已经有专门针对PowerShell进行美化的工具。

使用 Oh My Posh

了解Oh My Posh

Oh My Posh 是什么？

我们已经安装了Windows 终端，可以使用选项卡、壁纸等设置，获得美观的外壳；使用PowerShell可以获得更加强大的命令行功能；但在很多特定场景下，默认的命令行界面能够显示的信息太少了，工作效率不高。
比如，在使用Git时，我们希望直观地看到当前所处的分支，以及更改、暂存、提交状态，我们来看看下面2张图的区别。

如 ·图7· 所示，使用Windows 终端调用PowerShell，在普通状态下，几乎无法直观地获得任何信息。想要获得信息，就得输入各种繁琐的命令。再看看在Oh My Posh加持下的效果：

如 ·图8· 所示，效果不言而喻。使用了Oh My Posh，不仅可以显示当前 Git 分支名称，还能显示更改、暂存、提交的状态，甚至能识别出当前项目采用的是 Node.js 环境及其版本号18.16.1。这种将信息直接附加在提示符上的方式，省去了很多用命令查信息的时间。
