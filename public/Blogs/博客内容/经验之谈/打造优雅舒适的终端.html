为什么要美化终端？

  1. Windows以前的默认终端太丑了，如下图：

    如 图1 所示，Windows的官方终端目前有2个：

    • Windows 控制台主机传统
    • Windows 终端现代

  2. Windows 控制台主机，也就是我们常说的 cmd命令提示符。

    • 什么是 命令提示符 ？

      命令提示符是大多数Windows操作系统中可用的命令行解释器应用程序，用于执行输入的命令。

      提醒

      命令提示符有时被错误地称为“DOS提示符” 或 MS-DOS。命令提示符是一个Windows程序，它模拟MS-DOS中的许多命令行功能，但它不是MS-DOS。

    • 命令提示符 有什么缺点？

      • 界面丑陋
      • 缺乏自定义选项
      • 太过古老
      • 功能不够强大

  3. Windows 终端

    • 什么是 Windows 终端 ？

      Windows 终端 首发于2019年2月，由微软公司研发，是一款面向命令行用户的新型、现代化、功能丰富、高效的终端应用程序。它包括Windows命令行社区最常要求的许多功能，包括对选项卡、富文本、全球化、可配置性、主题化和样式化等的支持。

    • Windows 终端 有什么优点？

      • 免费开源
      • 微软公司背书，GitHub社区高度参与，性能、支持有保障
      • 标签化，可拆分
      • 高度自定义
      • 可聚合各种命令行工具
      • 支持富文本

  4. 即使是现代化的 Windows 终端，调用的也是其它的命令行工具，而这些命令行工具默认都只能显示枯燥的文本。但是，有的命令行工具可以高度自定义，可以产生丰富的视觉效果，又能提供丰富的信息，比如 PowerShell。

  • 安装 Oh My Posh

    • Oh My Posh 是用来干吗的？

      我们已经使用了Windows 终端，可以使用选项卡、壁纸等设置，获得美观的外壳；使用PowerShell可以获得更加强大的命令行功能，但是在很多特定场景下，默认的命令行界面能够显示的信息太少了，工作效率不高。
