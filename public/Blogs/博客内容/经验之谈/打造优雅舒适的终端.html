  • 为什么要美化终端？

    • Windows 默认终端太丑了。

      如图所示，Windows的官方终端目前主要有2个：

      • Windows 控制台主机传统
      • Windows 终端现代

    • 先来看看 Windows控制台主机 ，也就是我们常说的 cmd命令提示符 ：

      📢 什么是 命令提示符 ？

      现在的Windows系统，都是拥有用户界面的，可以依靠鼠标进行操作。但上世纪90年代前期以及更早以前，流行的是DOS操作系统，这个系统没有用户界面，只能依靠键盘输入文本命令进行操作界面如同上图。 后来出现了Windows操作系统，但是当时DOS依然被大量使用，且文本命令本身是刚需，因此在Windows中出现了一个名为命令提示符的程序，界面模仿了DOS操作系统。

      📢 命令提示符 为何被程序员吐槽？

      • 界面丑陋。
      • 可定制性太低。
      • 太过古老。
      • 功能不够强大。

    • 再来看看 Windows终端 Windows Terminal ：

      📢 什么是 Windows终端 ？

      Windows 终端 首发于2019年2月，由微软公司研发，是一款面向命令行用户的新型、现代化、功能丰富、高效的终端应用程序。它包括Windows命令行社区最常要求的许多功能，包括对选项卡、富文本、全球化、可配置性、主题化和样式化等的支持。

      📢 Windows终端 有何优点？

      • 免费开源。
      • 微软公司背书，GitHub社区高度参与，性能、支持有保障。
      • 标签化。
      • 大量可定制选项。
      • 可聚合其它各种终端。
      • 富文本。

    • 两张不同终端的截图，是不是给人的观感差别很大？爱美之心，人皆有之。

    • 安装 Windows终端 。

  • 安装 Oh My Posh

    • Oh My Posh 是用来干吗的？

      我们已经使用了Windows 终端，可以使用选项卡、壁纸等设置，获得美观的外壳；使用PowerShell可以获得更加强大的命令行功能，但是在很多特定场景下，默认的命令行界面能够显示的信息太少了，工作效率不高。
